Troubleshooting Steps

  1. Ensure the user.js file is in your Firefox profile directory
  2. The file should be directly in the profile folder, not in any subdirectory
  3. Restart Firefox completely (close all windows and start again)
  4. Try downloading and applying the settings again

Finding Your Profile Directory

  1. 1. Open a new tab in Firefox
  2. 2. Type about:support in the address bar
  3. 3. Look for "Profile Directory" and click "Open Directory"
  4. 4. Place the user.js file in this folder
